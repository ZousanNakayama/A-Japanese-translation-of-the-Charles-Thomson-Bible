# LXX(Septuagint)日本語訳プロジェクト
このレポジトリは、ギリシア語七十人訳聖書（Septuagint, LXX）を日本語で無料で読めるようにするための翻訳プロジェクトです。

七十人訳聖書は、紀元前3世紀頃にすでに存在したとされるギリシア語の旧約聖書です。
七十人訳聖書は、現在主流のマソラ本文とは内容が多くの点で異なっていて研究価値が高い一方で、日本語で読めるものはほとんどありません。
本プロジェクトでは、世界で初めて七十人訳聖書を英語に翻訳した、Charles Thomsonの英語聖書（Charles Thomson Bible, 1808, Public Domain）を日本語訳し、無料提供する試みです。
翻訳に当たっては、逐語訳を重視し、トムソン訳（Thomson’s Translation）での単語の意味が読者に透けて見えるような訳を目指しています。

## 📖 Charles Thomson Bible とは
**Charles Thomson Bible（1808）** は、ギリシア語七十人訳聖書（Septuagint, LXX）を  
英語へ忠実に翻訳した最初期の聖書です。

- 当時の英語圏ではヘブライ語原典からの翻訳が主流  
- 七十人訳に基づく英語訳はほとんど存在しなかった  
- Thomson は約 20 年をかけて原文を精査し、過度な意訳を避けた翻訳を完成  
- 1808 年版は **著作権が完全に消滅しており Public Domain**
