#  LXX（Septuagint）日本語訳プロジェクト
このレポジトリは、ギリシア語七十人訳聖書（Septuagint, LXX）を日本語で無料公開することを目的とした翻訳プロジェクトです。

七十人訳聖書は、紀元前1世紀ごろには広く用いられていたギリシア語訳の旧約聖書で、ユダヤ人や後のキリスト教徒に受け入れられていました。現在主流のマソラ本文とは多くの点で内容が異なっており、信仰的にも学術的にも重要な価値を持っています。一方で、日本語訳は限られており、入手は容易ではありません。

本プロジェクトでは、世界で初めて七十人訳聖書を英語に翻訳したチャールズ・トムソン（Charles Thomson）による1808年の英語訳聖書（Charles Thomson Bible, パブリックドメイン）をもとに、日本語訳を行い無料で提供することを目指しています。

翻訳にあたっては逐語訳を重視し、トムソン訳における語の意味やニュアンスが読者に伝わるような訳文を心がけています。

## 📖 Charles Thomson Bible とは
**Charles Thomson Bible（1808）** は、アメリカ合衆国建国の父の一人、チャールズ・トムソンがギリシア語七十人訳聖書（Septuagint, LXX）から直接英語に忠実に翻訳した歴史的に重要な聖書です。  
この翻訳聖書の主な特徴は以下の通りです。
- 4世紀のバチカン写本を反映した、17世紀の印刷本を底本とする
- Thomson は約 20 年をかけて原文を精査し、過度な意訳を避けた翻訳を完成  
- 1808 年版は **著作権が完全に消滅しており Public Domain**となっている
