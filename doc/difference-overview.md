# 📘 LXX と MT の本文差異ガイド  
この文書は、七十人訳聖書（LXX）とマソラ本文（MT）の間に存在する本文の違いを、  
**どこから読むべきか**  
**どの書に差異が多いのか**  
**どの箇所が神学的に重要なのか**  
という観点から整理したガイドです。

LXX と MT は同じ旧約聖書の伝統ですが、書によっては  
- 章の順序  
- 数値  
- 語彙  
- 神学的ニュアンス  
が大きく異なる場合があります。

このガイドでは、以下の3つの観点から優先度を分類しています。

---

# 🅐 A：本文の差異が大きい書  
**（構造・数値・語彙の違いが多く、比較する価値が高い書）**

これらの書は、LXX と MT の本文が大きく異なるため、  
**比較翻訳の優先度が最も高い** と言えます。

### 1. エレミヤ書  
- LXX は MT より約 1/8 短い  
- 章の順序が大きく異なる  
- 死海文書が LXX に近い  
→ 旧約で最も差異が大きい書

### 2. サムエル記（1–2）  
- 数値・固有名詞・物語の細部が異なる  
- ゴリアテの身長など有名な差異

### 3. 列王記（1–2）  
- 王の在位年数・年代が異なる  
- 系図の数字の違い

### 4. 申命記  
- 十戒の語彙差が大きい  
- 安息日の理由が異なる（創造 vs 解放）

### 5. 出エジプト記  
- 十戒の語彙・順序の違い  
- 奴隷法・過失致死の規定の訳語差

### 6. 民数記  
- 人口・儀式量など数値の違いが多い

### 7. レビ記  
- 動物名・儀式語彙の違い  
- 清浄規定の訳語差

### 8. イザヤ書  
- 語彙差は多いが構造差は少ない  
- 神学的には重要（後述）

---

# 🅑 B：神学的に重要な差異がある書  
**（新約引用・メシア預言など、特定の箇所が非常に重要）**

書全体の差異は大きくなくても、  
**特定の章・節が神学的に極めて重要** な書です。

### イザヤ書  
- 7:14（若い女性 vs 処女）  
- 53章（苦難の僕）  
→ キリスト論の核心

### 詩篇  
- 22:16（手と足を刺した）  
- 40:6（体を備えた）  
→ 新約が LXX を引用

### アモス書  
- 9:11–12（エドム vs 人々）  
→ 使徒15章で引用される重要箇所

### ミカ書  
- 5章（ベツレヘムの預言）

### 創世記  
- 4:7（罪の性質）  
- 47:31（杖の頭に寄りかかった：ヘブル書引用）

### 出エジプト記  
- 1:5（70人 vs 75人：使徒7章）

---

# 🅒 C：実際に守るべき規定が多い書  
**（生活規範・倫理規範として現代でも重要）**

信仰実践の観点から、  
**現代でも直接的に意味を持つ規定が多い書** です。

### レビ記  
- 清浄規定・食物規定・性倫理  
- LXX/MT の語彙差が誤解につながる

### 出エジプト記20章（十戒）  
- 偶像禁止・安息日・倫理規範の基礎

### 申命記5章（十戒の再掲）  
- 語彙差が大きく、理解に影響

### 申命記全体  
- 社会法・倫理法・契約更新

### 民数記の一部  
- 誓願・相続など実務的規定

---

# 🧭 推奨される読み進め方  
3つの観点を統合すると、以下の順番が最も効率的です。

## 1. **C：実際に守るべき規定（十戒・レビ記）**  
- 出エジプト記20章  
- 申命記5章  
- レビ記  

## 2. **B：神学的に重要な章（イザヤ7・53、詩篇22など）**  
- イザヤ7章  
- イザヤ53章  
- 詩篇22篇  
- アモス9章  
- ミカ5章  

## 3. **A：本文差異が大きい書（エレミヤ・サムエル・列王記）**  
- エレミヤ書  
- サムエル記  
- 列王記  

---

# 📌 まとめ  
- **A**：差異が大きい書 → 比較翻訳の価値が高い  
- **B**：神学的に重要な差異 → 新約引用・メシア預言  
- **C**：実際に守るべき規定 → 生活規範として重要  

このガイドは、LXX と MT の違いを理解しながら、  
どの書から読むべきかを判断するための道しるべとなることを目的としています。
